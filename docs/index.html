Privacy first

Nothing leaves your Mac.

Every part of HushScribe runs locally. There are no accounts, no servers, and no data collection. By design.

  • 🎙️
    Audio never touches a serverTranscription runs on-device via Apple Silicon. Your voice stays on your computer, always.
  • 🤖
    AI that stays localSummaries run on-device via Qwen3, Gemma 3, or Apple's NaturalLanguage framework — no external API, no API key, no network call.
  • 📁
    You own the outputTranscripts are plain .md files in a folder you choose. No proprietary format, no lock-in.
  • 🚫
    No telemetryNo analytics, no crash reporting, no usage tracking. Zero network requests at runtime.
  • 🖥️
    Hidden from screen sharingThe app window is invisible to screen recording by default — it won't appear on your call.
  • 🔓
    Open sourceFull source on GitHub. Read every line, build it yourself, or contribute. No black boxes.
Features

Built for people who want clean, structured meeting notes without giving up control of their data.

  • 🌍
    Multilingual transcriptionParakeet-TDT v3 — 25 European languages, auto-detected, runs on Apple Silicon's Neural Engine.
  • 🧠
    Multiple transcription modelsParakeet (default), WhisperKit Base, WhisperKit Large v3, or Apple Speech. All on-device.
  • 📞
    Call CaptureCaptures mic + system audio. Detects your conferencing app (Teams, Zoom, Slack…) and filters to just that source.
  • 🎤
    Voice Memo modeMic-only recording saved to a separate folder — voice notes never mix with meeting transcripts.
  • ✨
    On-device AI summaryOpen any transcript and click Generate Summary. Qwen3 0.6B, Gemma 3 1B, or Apple NL — all local, no API key. Supports custom prompts.
  • 👥
    Speaker diarizationSplits remote audio into labelled speakers after the session. Assign real names before saving.
  • ⚡
    Auto-record meetingsDetects active calls via app presence + mic activity. Starts and stops automatically. Browser-based meetings not supported.
  • 📋
    Vault-native outputStructured .md with YAML frontmatter — drop it into Obsidian or any folder-based workflow.
  • 🔕
    Lives in the menu barNo dock icon. Runs quietly in the background, accessible from the menu bar at any time.
  • 📦
    Self-containedNo Ollama, no local server, no background daemons. Everything runs inside the app — just download and go.
  • 📤
    Export transcriptsExport any transcript as SRT subtitles or structured JSON directly from the Transcript Viewer.
Output

Structured notes, ready to use.

Every session produces a single .md file with frontmatter, a summary, and a timestamped transcript.

You can summarise sessions using local AI models and your own prompts, then export to .srt subtitles or .json for use in other tools.

Install

Up and running in seconds.

Requires macOS 26+ and Apple Silicon.

No account, no license, no recurring fees, no catch

Homebrew (recommended) 
brew tap drcursor/hushscribe https://github.com/drcursor/HushScribe
brew install --cask hushscribe

To update: brew upgrade --cask hushscribe

Manual install
Download the DMG, open it, and drag HushScribe to /Applications.
